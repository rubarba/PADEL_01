# PadelCost VR26

Ficheiros preparados para GitHub Pages.

## Publicar

1. Sobe o conteudo desta pasta para um repositório.
2. Usa `index.html` como ficheiro principal.
3. Mantem `banner-vr16.jpg`, `logo-light-vr8.svg` e `logo-dark-vr8.svg` na raiz.

## Ficheiros

- `index.html`: versao pronta para publicar
- `site-final-padelcost-vr26.html`: copia da versao original
- `banner-vr16.jpg`: imagem do banner
- `logo-light-vr8.svg`: logo light
- `logo-dark-vr8.svg`: logo dark
